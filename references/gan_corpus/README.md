# GAN reference corpus

Eight past UCL BASc Final Year Dissertations (BASC0024) used as the reference set for an essay-iter GAN-style adversarial reviewer. The reviewer is shown the target dissertation anonymously alongside these references and tries to guess which is the target. References must match the target's cohort register so the reviewer cannot pick by surface mismatch.

All eight are 1sts (the source corpus is the curated Moodle exemplars page, which only posts 1sts). All 9.4k to 10.0k words, anonymous candidate codes only.

## Mix

- 4 technical (NLP, ML, AI/CS, theoretical CS)
- 4 non-technical (cultural heritage, law, philosophy, history)
- 3 with explicit LLM/transformer-era keywords (BERT, GPT-3, transformers, language model)
- Year spread: 2018, 2019, 2020 x3, 2021, 2022 x2

## Caveat

Corpus tops at 2022, so post-ChatGPT-mainstream register is absent. If the GAN reviewer ends up primed on early-2020s register, soften 2024+ paper citations in the target draft to match.
